About Typemania

This website was created as a personal learning project, exploring various aspects of web development. While at times frustrating, creating Typemania.net has been an educational and rewarding experience, expanding my knowledge in certain areas and introducing me to entirely new technologies in others.

Creating a website backend in Python (Flask)

I learned about routing, app structuring, handling user registration and logins, working with database models, working with a templating engine, logging website activity, sending emails, handling cookies, creating an admin interface, among other things.

A little bit of JavaScript

The Type game is written in simple vanilla JavaScript. I also learned a thing or two about DOM manipulation and communicating with a website's backend.

Styling and Sass

I learned the joys of CSS preprocessors, specifically Sass. I learned that CSS is powerful and requires planning and restraint.

Responsive web design

I learned about the marketshare of small screens and how to make website design responsive. In tests, Typemania.net has looked okay down to about a 300 pixel viewport width.

European privacy and cookie laws

Handling user email addresses and tracking website usage required some research into GDPR compliance and cookie consent law. Aside from difficulty in interpreting regulations, implementation of compliance with EU law proved a fun technical challenge.

Sentence generation

Game sentences are generated by words stored in a database transformed and combined into sentences by a script. At the moment, the sentence generating script has no semantic component and sentences therefore often don't make much sense. This might be addressed in a future update but for the time being sentences are often weird.

For the sentence generator, I learned how to serve data to the frontend through a simple API.

Planning and execution
