About Typemania

This website was created as a personal learning project, exploring various aspects of web development. Features include:

Creating a website backend in Python (Flask)

  • Registration / login system
  • Sending email
  • Allowing users to change login credentials, deleting user accounts
  • Personal user pages
  • High Score list
  • Logging errors and custom events
  • Sentence generation
  • Cookies handling
  • Admin page
  • Contact form
  • Custom error pages

Frontend and styling

  • The Type game is written in JavaScript
  • CSS preprocessed with Sass
  • HTML generated with Jinja

Responsive web design

The website's design is responsive and scales well down to 320px viewport width

European privacy and cookie laws

Handling user email addresses and tracking website usage required some research into GDPR compliance and cookie consent law.

Sentence generation

Sentences are generated by a pseudo-random sentence generator. Using so called sentence models, the script fetches random words with appropriate word type tags from a word database as well as from static assets. Words are transformed automatically if needed (e.g. pluralize nouns, conjugate verbs) and then inserted into the current sentence. Finally, the sentence is served to the game part of the application. At the moment, the sentence generating script has no semantic component and sentences therefore often don't make much sense. This might be addressed in a future update but for the time being sentences are often weird.
