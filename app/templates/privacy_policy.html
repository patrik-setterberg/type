Privacy policy

Typemania.net is a non-commercial personal project created for learning purposes by me, Patrik Setterberg. The application's full source code can be found on GitHub.

This website stores user data for some of its features, and tracks website usage. How Typemania.net handles users' personal information as well as tracks their use of the website is detailed on this page.

Email adresses and user accounts

Email addresses can be considered personally identifiable information. Registration of a user account on the Typemania.net website requires entering an email address.

Email addresses are only used to help users reset their passwords in the event they lose or forget them. There is no registration verification process and there are no newsletters.

Users can, at any time, change their login credentials and permanently delete their user accounts without delay (requires logging in) which erases all user information (such as email address, username) from the website database.

Usage tracking

Website usage is tracked with Google Analytics, a popular web analytics service. It can report user details such as their IP address, information about their web browsers, which pages of a website a user visits and how long they spend on each page, as well as other "diagnostic data".

For Typemania.net, only anonymized IP addresses are reported. This means that I can only really see what country users connect from. CONFIRM!!!??

Users may opt out of usage tracking at any time by visiting the Cookie Policy page, see below.

Typemania.net's purpose for collecting usage data are:

1) Learning. Google Analytics is widely used and I wanted to learn how to implement and use it.

2) Personal curiosity. Or, for fun. I will not actually do anything with the usage data except occasionally look at it.

Cookies

This website uses cookies for essential website functionality as well as for usage tracking. For more information about cookies, please see the Typemania.net Cookie Policy.

Security of data

The application and its database are stored on a secure, widely used hosting platform.

I will never share user email addresses or any anonymized usage data with any third parties, except to comply with possible legal investigations or obligations.

Contact me

For any inquiries regarding Typemania.net's handling of personal data, use of cookies or anything else, please contact me.
